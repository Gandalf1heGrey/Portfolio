Présentation du projet

Développement d'une interface de gestion de personnages basée sur l'univers Dragon Ball. Ce projet visait à structurer une base de données complexe tout en proposant une interface utilisateur moderne.

Solution technique

J'ai opté pour une architecture PHP pour la logique métier, couplée à une base de données MySQL. L'accent a été mis sur la sécurité (hachage des mots de passe) et la cohérence relationnelle des données.
